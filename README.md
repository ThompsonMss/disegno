# Disegno

Uma aplicação mobile desenvolvida com React Native, consumindo o serviço do Firebase.

> Funcionalidades

- Cadastro de usuário.
- Realização de Login.
- Cadastrar Projetos.
- Cadastrar Processos
- Cadastrar Tarefas
- Visualização de Projetos.

> Bibliotecas

- react-navigation.
- styled-components.
- firebase.
- react-native-vector-icons
